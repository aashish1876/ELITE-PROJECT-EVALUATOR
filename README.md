# 🚀 ELITE EVALUATOR

### *Stop guessing. Start growing. Get a FAANG-level review in seconds.*

**ELITE EVALUATOR** is a high-performance, AI-driven analysis engine designed to put your code through the ultimate stress test. Powered by the **Google Gemini 1.5 Flash API**, it simulates a panel of elite experts—Senior Engineers, System Architects, and Technical Interviewers—to give you a brutally honest, deep-dive evaluation of your project.

![Project Preview](https://picsum.photos/seed/code/1200/600)

---

## ✨ Key Features

*   **🔍 Multi-Phase Deep Analysis**: Goes beyond simple linting. We analyze logic, structure, scalability, and innovation.
*   **📊 Strict Scoring System**: No participation trophies here. Get a realistic 1-10 score across 8 critical dimensions.
*   **🧠 Developer Profiling**: Understand your coding style. Are you a "Pragmatic Architect" or a "Creative Problem Solver"?
*   **🎙️ AI Voice Feedback**: Listen to your evaluation with built-in Text-to-Speech capabilities.
*   **⚡ Interactive Interviewer**: Face a simulated FAANG-level technical interview based specifically on *your* code.
*   **📈 Visual Insights**: Beautiful radar charts powered by Recharts to visualize your project's strengths and weaknesses.
*   **🎨 Modern UI/UX**: A sleek, dark-themed interface built with Tailwind CSS and Framer Motion for smooth, professional interactions.

## 🛠️ Tech Stack

*   **Frontend**: React 19 + TypeScript
*   **Styling**: Tailwind CSS 4
*   **Animations**: Motion (Framer Motion)

*   **Charts**: Recharts

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/your-username/elite-project-evaluator.git
cd elite-project-evaluator
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Set Up Environment Variables
Create a `.env` file in the root directory and add your Gemini API Key:
```env
VITE_GEMINI_API_KEY=your_api_key_here
```
*(Get your key at [Google AI Studio](https://aistudio.google.com/app/apikey))*

### 4. Run the Development Server
```bash
npm run dev
```
The app will be running at `http://localhost:3000`.

---

## 🌍 Deployment

This project is optimized for **Vercel**. 

1. Push your code to GitHub.
2. Import the project into Vercel.
3. Add `GEMINI_API_KEY` to your Vercel Environment Variables.
4. Deploy!
